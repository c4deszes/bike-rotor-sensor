Speed Estimator

Calculation

WheelInch = 30 inch
BikeSpeed = 50km/h
Ticks = 64
WheelDiameter = WheelInch * 2.54 cm / 100 = 0.762 m
WheelCirc = WheelDiameter * Pi = 2.3939 m
RotationPerSec = (BikeSpeed * 1000 / 3600) / WheelCirc = 13.888 m/s / 2.3939 m = 5.8018 rps
TickPeriod = 1 / (RotationPerSec * Ticks) = 0.00269 s
